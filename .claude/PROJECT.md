# MOEX Cache Service

## Описание проекта
Надежный кэш-сервис для данных Московской биржи с целью обеспечения стабильной работы аналитики инвестиционного портфеля.

## Проблема
- API MOEX иногда недоступен, что ломает аналитику в Google Sheets
- Нужен единый источник данных для всех инвестиционных инструментов
- Требуется автоматическое обновление данных на регулярной основе

## Источники данных
1. **MOEX API** - котировки акций и облигаций (гос/корп) в формате CSV
2. **Google Finance** - курсы валют
3. **Dohod.ru** - прогнозируемые дивиденды (парсинг сайта)

## Целевая интеграция
- **Google Sheets** - автоматическое обновление вкладки "data" через API эндпоинты

## Архитектура

### Модели данных
- `Stock` - акции MOEX
- `Bond` - облигации (государственные и корпоративные)
- `Currency` - курсы валют
- `Dividend` - прогнозы дивидендов

### Компоненты системы
- **API Controllers** - CSV эндпоинты для Google Sheets
- **Background Jobs** - ежедневная синхронизация данных (Solid Queue)
- **Services** - клиенты для внешних API и парсеры
- **Cache Layer** - Solid Cache для надежности

### Обновление данных
- Частота: **1 раз в сутки**
- Время: утром (6-7 утра)
- При недоступности источника: отдача последних кэшированных данных

### API эндпоинты
- `/stocks.csv` - все акции MOEX
- `/bonds.csv` - все облигации MOEX  
- `/currencies.csv` - курсы валют
- `/dividends.csv` - прогнозы дивидендов

## Технический стек
- **Rails 8.1** (API-only)
- **PostgreSQL** - основная БД
- **Solid Cache** - кэширование
- **Solid Queue** - фоновые задачи
- **Kamal** - деплой

## Подход к данным
- Загружаем **все доступные** акции и облигации MOEX
- Фильтрация на стороне клиента (Google Sheets)
- Хранение только **текущих** актуальных данных (без истории)
