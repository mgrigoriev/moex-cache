# MOEX Cache Service - Claude Code Documentation

## Проект
Надежный кэш-сервис для данных Московской биржи с интеграцией в Google Sheets.

## Документация проекта
- [PROJECT.md](PROJECT.md) - полное описание проекта, архитектура, технический стек
- [PLAN.md](PLAN.md) - план разработки по фазам, текущий MVP на акциях

## Текущий статус
🎯 **MVP: Реализация кэширования акций (Stocks)**

### Следующие шаги
1. Исследование MOEX API для получения данных акций
2. Создание модели Stock и миграции
3. Реализация сервиса для синхронизации с MOEX
4. API эндпоинт `/stocks.csv` для Google Sheets

## Ключевые решения
- Rails 8.1 API-only приложение
- PostgreSQL + Solid Cache + Solid Queue
- Обновление данных: 1 раз в сутки
- Без хранения исторических данных
- Фокус на надежности при недоступности внешних API

## Команды для разработки
```bash
# Запуск приложения
bin/dev

# Миграции
bin/rails db:migrate

# Консоль
bin/rails console

# Тесты
bin/ci
```
